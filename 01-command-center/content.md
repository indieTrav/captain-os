## How to Use

**When you create a new content file:**

1. Save the file in the right folder (per the Output Folder Map in CLAUDE.md).
2. Add an entry under the right section using the format: `` `STATUS` — Title ``. Use the file's H1 as the Title.
3. If it's part of a repurpose batch, also add the bundle to the **Source Bundles** table (or bump the file count).

**When status changes:**

- Passed anti-slop / voice check → `REVIEWED`
- Assigned a publish date → `SCHEDULED`
- Live / sent → `PUBLISHED`
- Retired → `ARCHIVED`

**Weekly:**

- Bump the **Last updated** date.
- Refresh the **At a Glance** counts.
