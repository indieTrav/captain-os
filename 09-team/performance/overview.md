# Performance — How This Folder Works

*One file per person. Quarterly cadence on reviews. Append-only 1:1 logs.*

---

## File pattern

- `people/[person-slug].md` — evergreen file per team member. Role, comp, goals, growth areas, history.
- `1-1s/[person-slug]-log.md` — running 1:1 notes. Newest on top. Never delete entries.
- `reviews/[YYYY-Q#]-[person-slug].md` — formal quarterly performance review.

---

## Cadence

| Artifact | When | Owner |
|----------|------|-------|
| 1:1 notes | Weekly or bi-weekly | Manager logs, person reads |
| People file | After major changes (role, comp, goals) | Manager |
| Quarterly review | End of each quarter | Manager drafts, person responds |

---

## Principles

- 1:1s are for the report, not the manager.
- No surprises in reviews — feedback flows continuously.
- Document decisions, not just discussions.
